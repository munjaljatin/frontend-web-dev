Frontend Developement is the Developement of the graphical user interface of a website throught the use of HTML, CSS, and Javascript, so that the user can view and interact with that particular website.

  • HTML: HTML is standard Markup Language for documents designed to be displayed on a web browser.


  • CSS: CSS stand for Cascading Style Sheet. CSS is style sheet Language used for describing the presentation of document written in a Markup language such as HTML or XML.

    2. Bootstrap

    3. Tailwind CSS

    4. SASS

  • Javascript: Javascript (JS) is a lightweight interpreted programming language with first-class-function. It is most well-known as the scripting language for the web pages.

    1. Vanilla JS

    2. JS Library

    3. JS Frameworks

    4. Package Manager

    5. Typescript
